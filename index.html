special-draw

special-draw is a very simple way to implement advanced graphical effects in your Pebble apps. Some examples of possible modifiers are:

Rotation Transparency Dithering Mirroring [coming soon] Opacity Masks [coming soon] Gradients [coming soon]

Usage example 

static void layer_update_proc(Layer *layer, GContext *ctx) {
    graphics_draw_rect(ctx, GRect(32, 54, 80, 60));
    GSpecialSession * session =
        graphics_context_begin_special_draw(ctx);
    graphics_context_set_stroke_color(ctx, GColorBlack);
​
    graphics_draw_rect(ctx, GRect(38, 60, 80, 60));
​
    graphics_context_special_session_add_modifier(session,
        graphics_special_draw_create_opacity_modifier(GOpacity2));
    graphics_context_special_session_add_modifier(session,
        graphics_special_draw_create_rotation_modifier(TRIG_MAX_ANGLE/8));
​
    graphics_context_end_special_draw(session);
}

Philosophy

  • special-draw should let you draw anything.
  • special-draw should be easy to extend.
  • special-draw should let you chain modifiers.

special-draw is an MIT-licensed package for Pebble. Modifiers can also be installed as packages, and there is a very simple API that lets you implement custom modifiers. For more information about the app and modifier APIs, see the README.
